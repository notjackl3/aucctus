# aucctus-repo-setup

## Use when
- Setting up or validating the Aucctus frontend local environment.

## Workflow
1. Install dependencies: `cd aucctus && npm install`.
2. Run checks: `npm run type-check && npm run lint`.
3. Start dev server: `npm run dev`.
4. Confirm API/environment configuration needed for local development.
